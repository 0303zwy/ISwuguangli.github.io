hexo+github搭建个人博客

废了几个小时，终于搭好个人博客了。本人采用的是hexo+github搭建，使用的主题是jacman

工具准备

  • 安装Node.js及配置Node.js环境

  • 访问nodejs官网下载最新版本，开始安装时点击add path，将npm和node路径添加系统路径。接下来一路next安装即可
  • 安装完毕，进入cmd，分别输入node -v与npm -v能出现相应的版本号即成功

    • 全局安装hexo

  • 进入cmd，输入命令:npm install hexo -g全局安装hexo
  • 输入hexo -v出现相应版本即成功

    • 安装git及配置git环境

  • 访问git官网下载最新版本，一路next即可
  • 安装完毕，进入cmd，输入git --version能出现版本号即成功
  • git是一个非常好用的代码管理工具，若想详细了解git可访问廖雪峰git教程

    • 注册github账户，并新建一个respository

  • respository的名字为：你的github账户名.github.io
  • 类型为public
  • 勾选Initialize this repository with a README

    安装hexo并部署到github

    在自己的电脑上找一个合适的位置，建一个文件夹，下面的操作都在这个文件下进行。我是在E盘根目录下建了一个叫blog_hexo的文件夹。

    • 初始化hexo并在本地运行

  • 打开cmd，进入刚刚建的文件下（下面的所有操作都在这个文件下进行）
    • cmd中输入命令E:
    • cd blog_hexo
  • 初始化hexo
    • 输入hexo init
    • cmd中出现Start blogging with Hexo!择安装成功
    • npm install安装需要的组件

  • 在本地运行

    • hexo g
    • 开启服务hexo s，这时候会给你一个端口，访问这个端口就可以看到本地的效果了

    • 部署到github上

      若是第一次使用git，请设置git用户信息，并将公钥添加到github上
      下面的所有操作仍进入cmd，在blog_hexo目录下进行

  • 打开blog_hexo文件夹下的_config.yml,为了方便，站点配置文件 特指这个文件，打开并找到deploy:这一行，加上

    • repository: git@github.com:你的用户名/你的用户名.github.io.git
    • branch: master
  • 安装扩展，输入npm install hexo-deployer-git --save
  • 新建博客，输入hexo new post "博客名",这是在_post文件夹上能看到刚刚创建的文件
  • 编辑好文章后，输入 hexo d -g,这时候就完成了，在浏览器中访问：https://你的用户名.github.io/就可以看到你写的文章了。

    使用jacman主题

  • 在blog_hexo文件夹上右键，选择Git Bash Here
  • 进入gitbash后，输入git clone https://github.com/wuchong/jacman.git themes/jacman，克隆成功后，在blog_hexo下的themes目录下可以看到克隆下的主题仓库
  • 修改站点配置文件，修改主题为theme为：jacman
  • 站点目录下还可配置博客名等
  • 关于jackman的配置指南请参考jackman作者博客
