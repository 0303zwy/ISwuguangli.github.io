哈希表的基本概念

官方定义：

简单理解就是，哈希表就是用唯一标识查找，如学号可代表学生，用学号查找学生就节省时间的多

java中hashmap常用方法

  • 初始化
    • HashMap<Integer, Integer> map=new HashMap<>();
  • 常用方法
    • map.put(key,value)
    • map.get(key.value)
    • map.isEmpty()//判断hashmap是否为空
    • map.containsKey(key)//判断是否有这个key
    • map.containsValue(Value)//判断是否有这个Value
    • map.remove(key)//删除key下面的value
    • map.replace(key,value)//替换
    • map.values()//显示所有的value
